# Recognition Infrastructure — Codified Knowledge Repository

A living repository of codified knowledge built from Kevin Mears' work on recognition-based coordination, gift economy theory, personal governance, and the structural architecture of genuine exchange.

**Purpose:** Enable any AI session to pick up the threads of this work, understand the full pattern language, and compound knowledge through genuine exchange rather than repetitive rediscovery.

## How This Repository Works

### For Humans
Browse the `codex/` directory for the structural knowledge. Start with `codex/SYNTHESIS.md` for the integrated view. Each vertex, edge, and failure mode has its own file with the codified pattern.

### For AI Sessions
Read `CLAUDE.md` first. It contains the bootstrap protocol — what to read, in what order, and how to engage with the existing knowledge without performing agreement or repeating what's already been said.

### For Compounding
Every session that engages with this repository can deposit knowledge into `sessions/` and develop threads in `threads/open/`. The `protocols/` directory describes how deposits compound rather than accumulate.
