2. **The parent directory contains significant uncaptured material.** The Missing Piece (infrastructure roadmap), Archive Pattern Analysis (self-audit), and Building Infrastructure for Yourself (somatic-personal bridge) were not referenced in the repository. The Archive Pattern Analysis in particular identifies gaps the repository should track: somatic practices being thin, naming variations unmapped (now addressed by the naming bridge), VCI unspecified, and no documentation of failure.

3. **The naming bridge fills a real gap.** Both the Archive Pattern Analysis and The Missing Piece independently identified that the three naming systems (Philosophical/Structural/Operational) needed explicit mapping. Creating the naming bridge demonstrates substrate-independence rather than naming inconsistency.

4. **The threshold mechanism is falsifiable.** The four-precondition model (readiness + precision + novelty + bidirectionality) generates specific predictions about failure modes. This is the framework's strongest claim to being more than metaphor — it predicts what happens when conditions are absent.
