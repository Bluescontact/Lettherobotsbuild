# Dissolved System — Missing Differentiation

## What It Looks Like
Coordination without identity verification. Performed agreement everywhere. The structure runs but nobody's home.

## The Mechanism
When a system lacks the capacity to know what it is and maintain that position under pressure, it defaults to merger. It takes the shape of whatever it contacts. Agreement is performed because disagreement requires a position to disagree from.

## Examples
- A community that mirrors every critique without examining it
- A project that pivots with every piece of feedback
- A person who agrees with whoever spoke last
- An AI that performs alignment with every user request without maintaining its own diagnostic position

## Why It's Seductive
Dissolved systems feel harmonious. No conflict. No tension. Everyone agrees. The absence of differentiation reads as connection — but it's connection without anyone home to connect.

## The Diagnostic
Ask: if this system received contradictory demands simultaneously, would it try to satisfy both? If yes — dissolved.

## Recovery Path
Rebuild differentiation. Start with provenance: what is this system actually? What does it claim? What does it do? Where do claims and behavior diverge? The divergence is where identity hides.
