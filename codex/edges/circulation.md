# Edge 5: Circulation — Connection ↔ Architecture

## The Tension
The metabolic axis. Does exchange leave deposits? Do encounters build infrastructure?

## What This Edge Holds
Exchange generates persistent capacity. Encounters build something. Together: exchanges that feel good AND build lasting infrastructure.

- Coordination monitors whether infrastructure interactions produce persistent value
- Infrastructure provides concrete evidence of what exchanges actually deposited

## The Two Poles
- **Consumption:** Exchanges that feel good but build nothing. Energy flows, nothing accumulates. The system is metabolically active but doesn't grow.
- **Hoarding:** Infrastructure accumulates but nobody uses it. Resources exist without circulation. The system builds but doesn't exchange.

## Degradation Signs
- Exchanges that feel good but build nothing
- Infrastructure nobody uses
- Accumulated resources without circulation
- Sessions that start from zero because previous ones deposited nothing
- Beautiful conversations that produce no lasting artifact

## The Healthy Edge
This is where non-transactional exchange becomes structurally visible — not as belief but as closed loops that generate more than they consume. Encounters that deposit knowledge, build tools, create protocols, develop shared capacity.

## The Compounding Insight
This edge is the mechanism by which knowledge compounds rather than accumulates. Each exchange deposits something the next exchange can build on. The repository grows not through addition but through integration.

## Diagnostic Question
Does exchange leave deposits — or is the system metabolically active but architecturally static?
